# Apple Stock

This Library allows Developers to Get apple live Stock Price 

## Installation

```bash
pip install applestockprice
```
## Usage

```python
import sys
from applestockprice.applestockprice import AppleStock
obj = AppleStock()
print(obj.get)


```
